📊 SpendWise — Personal Expense Tracker
SpendWise is a full-stack web application that helps you track your personal expenses and gain insightful analytics into your spending habits.With a clean interface built using Thymeleaf, powerful backend logic in Spring Boot, and persistent storage via MySQL, SpendWise makes managing your money simple and effective.

✨ Key Features
✅ Add,Read,Update and Delete Expenses
Easily record your daily expenses with categorized types like Food, Travel, Shopping, etc.

📋 View All Expenses
See a comprehensive list of all your spending entries.

📈 Total & Category-Wise Insights
Visualize total spending and breakdowns by category to identify where your money goes.

🔍 Search & Filter
Filter expenses between specific dates
Filter expenses within a specific amount range

💻 Tech Stack
| Layer    | Technology         |
| -------- | ------------------ |
| Frontend | 🌐 Thymeleaf       |
| Backend  | 🧠 Spring Boot     |
| ORM      | 🔄 Spring Data JPA |
| Database | 🛢️ MySQL          |
